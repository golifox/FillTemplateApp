# Приложение для автоматического создания WORD - шаблонов

## Вид шаблона
Текстовый файл любого текстового формата (.docx, .odt, .txt, .rtf), содержащий какой то формализованный документ. 
В этом документе часто изменяемые поля (ФИО, дата, количество дней для отпуска и.т.д) обозначены тэгами (в текущей программе символом '#') с названиями переменных.  
Например: 
> Заявление от #ФИО автора# для #ФИО получателя#



## Возможности 
* Открытие шаблона для редактирования переменных имен
* Открытие папки с шаблонами для вывода их в дереве (визуальном)
* Сохранение шаблона с измененными данными в виде отдельного документа
* Логирование действий пользователя в базу данных

## Features 
* Отправка созданного документа по e-mail
